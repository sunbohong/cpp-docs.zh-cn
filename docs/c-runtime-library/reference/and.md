# <a name="and"></a>和

&& 运算符的替代项。

## <a name="syntax"></a>语法

```C

#define and &&
```

## <a name="remarks"></a>备注

该宏产生运算符 &&。

## <a name="example"></a>示例

```cpp
// iso646_and.cpp
// compile with: /EHsc
#include <iostream>
#include <iso646.h>

int main( )
{
   using namespace std;
   bool a = true, b = false, result;

   boolalpha(cout);

   result= a && b;
   cout << result << endl;

   result= a and b;
   cout << result << endl;
}
```

```Output
false
false
```

## <a name="requirements"></a>要求

**标头：**\<iso646.h>
