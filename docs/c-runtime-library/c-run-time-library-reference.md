# <a name="c-runtime-library-reference"></a>C 运行时库引用

Microsoft 运行时库为 Microsoft Windows 操作系统的编程提供了例程。 这些例程自动执行许多不采用 C 和 C++ 语言提供的常见编程任务。

库中大部分例程的示例程序包含在独立参考主题中。
