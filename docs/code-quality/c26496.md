# <a name="c26496-use_const_for_variable"></a>C26496 USE_CONST_FOR_VARIABLE

> 仅分配一次变量 "*variable*"，将其标记为 `const` 。 

## <a name="see-also"></a>请参阅
[C++ Core Guidelines](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#con4-use-const-to-define-objects-with-values-that-do-not-change-after-construction)

## <a name="example"></a>示例
```cpp
int GetTheNumber();
int GiveMeTheNumber(int);

void function1()
{
    int theNumber = GetTheNumber(); // C26496, 'theNumber' is never assigned to again, so it can be marked as const
    std::cout << theNumber << '\n';
    
    GiveMeTheNumber(theNumber);
    // ...
}
```
