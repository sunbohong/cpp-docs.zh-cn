# <a name="compiler-warning-level-1-c4226"></a>编译器警告（等级 1）C4226

> 使用了非标准扩展： "*关键字*" 是过时的关键字

Visual C++ 的当前版本不使用此关键字。

此警告会自动提升为错误。
