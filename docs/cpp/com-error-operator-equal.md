**Microsoft 专用**

将现有 `_com_error` 对象赋给另一个对象。

## <a name="syntax"></a>语法

```
_com_error& operator = (
   const _com_error& that
) throw ( );
```

#### <a name="parameters"></a>parameters

*that*<br/>
`_com_error` 对象。
